## 🎨 Efeitos Visuais

### Sempre Ativos
- Partículas no fundo (se Particles.js carregar)
- Cursor customizado (desktop)
- Hover effects em todos os cards
- Tema claro/escuro

### Ao Rolar a Página
- Animações GSAP (se carregar)
- Efeitos 3D nos cards
- Barras de progresso animadas

## 🔧 Se Algo Não Funcionar

### Partículas não aparecem
- Verifique a conexão com internet (CDN)
- Abra o console (F12) e veja se há erros

### Animações não funcionam
- O conteúdo AINDA ESTARÁ VISÍVEL
- Apenas as animações não funcionarão
- Verifique se os CDNs estão carregando

### Cursor customizado não aparece
- Normal em dispositivos touch/mobile
- Funciona apenas em desktop com mouse

## 📱 Responsividade

- **Desktop**: Todos os efeitos ativos
- **Tablet**: Sem cursor customizado
- **Mobile**: Sem cursor, layout adaptado

## 🎯 Arquivos Principais

```
index.html          - HTML com inline styles para visibilidade
assets/css/style.css - Todos os estilos
```
